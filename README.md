# 하루브금

**하루브금**은 그누보드5에서 사용할 수 있는 시간표 기반 BGM 자동재생 플러그인입니다.

정해진 시간에 파일 음악 또는 YouTube 링크 음악을 재생하고, 관리자 공용 운영판과 순서표 모드를 통해 교회 예배, 방송, 행사, 매장, 학교 등의 현장 운영에도 활용할 수 있습니다.

## 주요 특징

* 그누보드5 로그인 연동
* 회원별 개인 BGM 시간표
* 관리자 전용 공용 운영판
* 공통 시간표 / 개인 시간표 지원
* 특정 시간 동안 재생
* 시간대 묶음 재생
* 파일 음악 + YouTube 링크 혼합 재생
* 정각 1회 세트 재생
* 순서표 모드 지원
* 교회 · 방송 · 행사 진행판 지원
* 회원별 하루브금 사용 ON/OFF 관리
* 재생 로그 및 설치 점검 기능 제공

## 이런 곳에 추천합니다

* 개인 루틴 BGM
* 교회 예배 전후 음악
* 방송 대기음
* 행사 진행 음악
* 매장 시간대별 BGM
* 학교/학원 알림 음악
* 커뮤니티 회원 전용 음악 서비스

## 설치 방법

압축을 해제한 뒤 `plugin/haru_bgm` 폴더를 그누보드5의 `plugin` 폴더 안에 업로드합니다.

```text
/그누보드루트/plugin/haru_bgm/
```

업로드 후 관리자 계정으로 로그인하고 아래 주소에 접속합니다.

```text
/plugin/haru_bgm/
```

필요한 DB 테이블은 첫 접속 시 자동 생성됩니다.

설치 상태는 아래 페이지에서 확인할 수 있습니다.

```text
/plugin/haru_bgm/admin/health.php
```

## 기본 사용 흐름

1. 관리자 계정으로 접속합니다.
2. 음악 파일 또는 YouTube 링크를 등록합니다.
3. 공통 시간표, 시간대 묶음, 순서표 중 원하는 방식을 설정합니다.
4. 공용 운영판 또는 회원용 플레이어에서 `음악 알림 켜기`를 누릅니다.
5. 정해진 시간에 브라우저에서 음악이 재생됩니다.

## 주의사항

브라우저 정책상 사용자의 클릭 없이 소리가 자동으로 재생되지 않을 수 있습니다.
처음 사용할 때는 반드시 `음악 알림 켜기` 버튼을 눌러야 합니다.

YouTube 영상은 임베드가 허용된 영상만 정상 재생됩니다.
현장 운영 전에는 반드시 미리듣기 또는 테스트 재생을 권장합니다.
