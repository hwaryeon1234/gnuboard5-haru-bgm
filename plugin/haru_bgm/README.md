# 하루브금 v1.13

그누보드5용 **시간표 자동재생 / 시간대 묶음 / 공용 운영판 / 순서표 진행** 플러그인입니다.

로그인한 회원 또는 관리자가 웹페이지를 열어둔 상태에서, 정해진 시간표에 맞춰 파일 음악 또는 YouTube 링크 음악을 재생할 수 있습니다.

---

## 주요 기능

### 1. 로그인 연동

- 그누보드5 회원 로그인 연동
- 비로그인 접근 차단
- 관리자 전용 운영판 제공
- 회원별 하루브금 사용 ON/OFF를 운영자가 지정 가능

### 2. 회원용 내 BGM

- 로그인 회원별 개인 시간표
- 개인 시간대 묶음
- 내 브라우저에서만 재생
- 볼륨 저장
- 오늘만 끄기
- 전체 정지

### 3. 관리자 공용 운영판

- 관리자만 접근 가능
- 모든 관리자에게 같은 공통 운영표 표시
- 공통 시간표 / 공통 시간대 묶음 기반 재생
- 교회, 방송, 행사, 매장, 학교용 운영에 적합

### 4. 공통 시간표

- 정각 1회 재생
- 특정 시간 동안 재생
- 시작 시간 / 종료 시간 설정
- 종료 시간까지 반복 재생
- 파일 음악 + YouTube 링크 혼합 세트 지원
- 정각 1회에서도 여러 곡 세트 재생 지원

### 5. 시간대 묶음

- 예: `09:30 ~ 10:00` 예배 전 묵상 BGM
- 여러 곡을 시간 안에서 순서대로 또는 랜덤 재생
- 반복 재생 가능
- 파일 음악과 YouTube 링크 혼합 가능
- 최대 100개 음악 관리

### 6. 순서표 모드

교회 예배, 방송, 행사 진행용입니다.

예시:

1. 예배 전 묵상 BGM
2. 입례송
3. 찬양 1
4. 찬양 2
5. 헌금송
6. 폐회송

지원 기능:

- 순서표 생성/수정
- 항목별 파일 음악 / YouTube 음악 연결
- 순서표 진행판
- 항목별 재생
- 전체 순서 테스트
- 전체 정지
- 실수방지 확인
- 모바일 조작 버튼 보정

### 7. YouTube 링크 지원

- YouTube URL 또는 영상 ID 등록 가능
- 여러 링크 한 번에 붙여넣기 가능
- 파일 음악과 YouTube 링크를 한 시간표에 같이 사용 가능
- YouTube 임베드 플레이어 기반 재생

주의:

- YouTube 영상은 임베드가 허용된 영상만 정상 재생됩니다.
- YouTube 정책상 브라우저에서 사용자가 먼저 소리 재생을 허용해야 합니다.
- 일부 영상은 저작권/임베드 제한으로 재생되지 않을 수 있습니다.

### 8. 음악 업로드

- mp3
- wav
- ogg
- m4a

지원 기능:

- 다중 업로드
- 드래그 앤 드롭 업로드
- 파일명 기반 자동 제목 등록
- 음악 보관함 관리
- 파일/YouTube 구분 표시

### 9. 재생 로그

- 자동재생
- 미리듣기
- 수동진행
- 성공/실패 기록
- 실패 메시지 기록
- 파일/YouTube 구분

## 설치 방법

압축을 풀어 그누보드5 루트에 업로드합니다.

```text
/plugin/haru_bgm/
```

예:

```text
그누보드 루트
├─ bbs/
├─ adm/
├─ plugin/
│  └─ haru_bgm/
└─ common.php
```

업로드 후 아래 주소로 접속합니다.

```text
/plugin/haru_bgm/
```

첫 접속 시 필요한 DB 테이블은 자동 생성됩니다.

## 운영 방식 추천

### 개인 루틴용

- 회원용 플레이어 사용
- 내 시간표 / 내 시간대 묶음 등록
- 브라우저에서 `음악 알림 켜기` 클릭 후 사용

### 교회 예배용

- 관리자 공용 운영판 사용
- 예배 전 BGM은 공통 시간대 묶음으로 구성
- 본 예배 진행은 순서표 모드 사용
- 진행자는 순서표 진행판에서 다음 곡을 수동 재생

### 방송/행사용

- 대기음은 특정 시간 동안 재생
- 방송/행사 본 진행은 순서표 모드 사용
- 전체 정지 / 실수방지 확인 사용 권장

### 매장/학교용

- 공통 시간표 또는 공통 시간대 묶음 사용
- 점심/저녁/마감 시간대별 BGM 구성
- 관리자 공용 운영판을 담당자 화면으로 사용

## 브라우저 자동재생 안내

대부분의 브라우저는 사용자의 클릭 없이 소리가 자동 재생되는 것을 막습니다.

따라서 처음 사용할 때는 반드시 화면의 버튼을 눌러야 합니다.

```text
음악 알림 켜기
```

이후 같은 브라우저에서는 저장된 설정에 따라 시간표 재생이 동작합니다.

---

## YouTube 사용 주의

YouTube 링크는 다음 형식을 지원합니다.

```text
https://youtu.be/영상ID
https://www.youtube.com/watch?v=영상ID
https://www.youtube.com/embed/영상ID
영상ID
```

단, 다음 경우 재생되지 않을 수 있습니다.

- 영상 소유자가 외부 임베드를 막은 경우
- 저작권 제한이 있는 경우
- 지역 제한이 있는 경우
- 영상이 삭제/비공개 처리된 경우
- 브라우저가 자동재생을 차단한 경우

현장 사용 전 반드시 미리듣기 또는 테스트 재생을 권장합니다.

업데이트 후 관리자 설치 점검 페이지에서 상태를 확인하세요.

```text

```

---

## 삭제 방법

플러그인 파일만 삭제하려면 아래 폴더를 삭제합니다.

```text
/plugin/haru_bgm/
```

업로드된 음악 파일은 기본적으로 다음 위치를 사용합니다.

```text
/data/haru_bgm/
```

DB 테이블까지 삭제하려면 직접 DB에서 `haru_bgm` 관련 테이블을 삭제해야 합니다.

주의: DB 테이블 삭제 시 시간표, 음악 정보, 순서표, 로그가 모두 사라집니다.

## 문제 해결

### 음악이 안 나와요

1. `음악 알림 켜기` 버튼을 눌렀는지 확인
2. 브라우저 음소거 여부 확인
3. 볼륨 설정 확인
4. 파일 경로/업로드 상태 확인
5. YouTube 영상의 임베드 가능 여부 확인
6. 관리자 설치 점검 페이지 확인

### YouTube만 안 나와요

1. 다른 YouTube 영상으로 테스트
2. 임베드 제한 영상인지 확인
3. 브라우저 콘솔 오류 확인
4. 네트워크 또는 광고차단 확장 프로그램 확인

### 특정 회원이 안 들어가져요

관리자 회원 사용설정에서 해당 회원이 OFF인지 확인합니다.

```text
/plugin/haru_bgm/admin/member_access.php
```

### 시간표가 안 보여요

1. 요일 설정 확인
2. 시작일/종료일 확인
3. 사용 여부 ON 확인
4. 회원용인지 공통용인지 확인
5. 관리자 공용 운영판은 공통 데이터만 표시됨

## 안내

하루브금은 웹브라우저 기반 재생 플러그인입니다.

서버가 모든 사용자 기기에 강제로 소리를 송출하는 방식이 아니라,  
각 사용자가 열어둔 브라우저에서 시간표에 맞춰 재생하는 방식입니다.

따라서 현장 운영용으로 사용할 경우, 담당자 기기에서 사전에 반드시 테스트해 주세요.
